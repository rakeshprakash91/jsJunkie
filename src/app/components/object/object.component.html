Object & Arrays

An object is a collection of properties, properties in the form of key value pair. The key is usually a string name, and the value might be anything from a simple number/boolean/string to a function. As we have seen in the datatype section, the object properties can be accessed using the dot(.) operator.

In javascript, both Arrays & Objects are of the data type Object.

 
        var a = [1,2,3];

        console.log(typeof a); // object

        var obj = {{'{'}} b : 1{{'}'}} 

        console.log(typeof obj); // object

The key difference between arrays & objects are, array is a collection of data of single data type i.e if it is a number array, all the elements would be numbers, or if it is a string array, all the elements will be string. However when it comes to objects, one property can have a number value, another one a string. 

    var arr = [1,2,3,4]; // all elements are numbers
    

    var obj = {{'{'}}
        x : 5,
        y : "hello world"
    {{'}'}}
Creating Arrays There are multiple ways to create an array 
      var arr = []; // empty array initialization
      var arr = [1,2,3,4]; // Initializing array with values
      var arr = new Array(); // instantiating using new keyword

Prefer the 1st or 2nd method, as it is faster.

Creating Objects Just like arrays, we have multiple ways to create an object as well 
        var obj = {{'{'}}{{'}'}}; // empty Object initialization
        var obj = {{'{'}} b : 1{{'}'}}; // initializing with value
        var obj = new Object();

Prefer the 1st or 2nd method, as it is faster.

Accessing Array Elements 
        var arr = [1,2,3,4];
        console.log(arr[0]); // 1

Array elements are accessed by their index. Index starts from 0. So to access the first element in the array, we write array name followed by square brackets and the number 0.
Accessing Object Properties 
        var obj = {{'{'}}
            x : 1,
            y : 2
        {{'}'}};
        console.log(obj.x); // 1
        console.log(obj['x']); // 1
Object properties are accessed by using the dot(.) operator. For instance if we wanted to access the property x, we would write object name dot(.) property key i.e obj.x . We can also access the property x using the square brackets, and passing in the key as a string i.e obj['x']
Adding new elements to arrays

Using push

        var arr = [1,2,3];
        arr.push(4);
        console.log(arr); // [1,2,3,4]
Using the push method, a new value can appended to the existing array.

Using Splice

        var arr = [1,2,3];
        // arrayName.splice(index, numberOfElementsToBeRemoved, newElemetsToBeAdded)
        arr.splice(0,0,0);
        console.log(arr); // [0,1,2,3]

As given above, the syntax for using splice to add new element is, the index in which the new values need to be entered, the second parameter is number of elements that need to be removed from the index specified previously, and finally the elements that need to be added to the array. Since we are intending to only add new elements, we will keep the 2nd parameter 0; The advantage of using splice is we can add a new element in any given index, we can add as many number of elements in one go. Also, if we wish to remove some elements and then add new elements, that is also possible. 
        var arr = [1,4,3];
        arr.splice(1,1,2);
        console.log(arr); // [1,2,3]

In the above code snippet, we removed the element in the 2nd position and replaced it with the value 2.

Using unshift

        var arr = [1,4,3];
        arr.unshift(0);
        console.log(arr); // [0,1,4,3]
The unshift method adds the parameters it has to the begining of the array.
Adding new properties to objects

Using dot(.)

        var obj = {{'{'}}{{'}'}};
        obj.a = 1;
        console.log(obj); // Object {{'{'}}a: 1{{'}'}}

The above code, initialized an empty object. Using the dot operator, we are adding a new property a to the objet obj. We can also have the property contain a function in the following way. 
        var obj = {{'{'}}{{'}'}};
        obj.a = function(){{'{'}}
            console.log('hello world');
        {{'}'}}
        obj.a(); // hello world
Deleting element from array

Using splice

        var arr = [1,2,3];
        arr.splice(1,1);
        console.log(arr); // [1,3]
As discussed above, in the splice method, if we give only the first 2 parameters, the elements will get removed or spliced from the array.

Using pop

        var arr = [1,2,3];
        arr.pop();
        console.log(arr); // [1,2]
The pop method would remove the last element in the array.

Using Shift

        var arr = [1,4,3];
        arr.shift();
        console.log(arr); // [4,3]
The Shift method, effectively moves the elements to the left by 1 place, effectively removing the first element in the process. It returns the element that was shifted.
Deleting property from object 
        var obj = {{'{'}}
            a : 1
        {{'}'}}
        delete obj.a;
        console.log(obj); // Object{{'{'}}{{'}'}}
To delete a property from an object, use the keyword delete, followed by the object name dot(.) property name.

Note

The data-type of arrays & objects can be identified by using instanceof

         var arr = [1,2,3];
        console.log(arr instanceof Array); // true 

        var obj = {{'{'}} b : 1{{'}'}};
        console.log(obj instanceof Object); // true


Do not use delete to delete elements from an array, it would leave the corresponding element position with undefined value, which means the size of the array won't reduce even though the element has been deleted.


We can use prototype to add new properties & Object.create() to create new objects. However since we are not covering OO concepts in here, we would be skipping it.
