Functions

The various types of functions in javascript are :

  • Named functions
  • Annonymous functions
  • IIFE

Functions are typically blocks of code, which are designed to perform a specific task.

Named functions The functions we have seen in the previous section is what is termed as named function. As the name suggests, named functions are those functions which have name. 
      function abc(){{'{'}}
        alert("Hello world");
      {{'}'}}
To invoke/execute the function, we will call it by its name & the paranthesis.
abc();// Hello World

Annonymous functions As the name suggests, annonymous functions are those functions which don't have a name. They are defined pretty much the same way as named functions, except they dont have a name. 
      function (){{'{'}}
        alert("Hello world");
      {{'}'}}
Annonymous functions as such cannot be invoked, since they dont have a name. So usually they are assigned to a variable, or defined as a callback function. 
    var x = function (){{'{'}}
        alert("Hello world");
    {{'}'}}
In the above code we are assigning the annonymous function to the variable x. Now that we have the annonymous function stored in variable x, we can invoke it just like a named function.
x(); // Hello World 
      setTimeout(function(){{'{'}}
        alert("Hello World");
      {{'}'}}, 1000)
What the above code does is, it executes the annonymous function after a period of 1sec(1000 milliseconds). In this context, we are using annonymous function as a callback function.

IIFE IIFE stands for Immediately Invoking Function Execution. IIFE is an annonymous function which is executed as soon as it is loaded in the memory, hence the name Immediately invoking functions. These types of functions are popularly found in plugins, for instance jQuery. We just import the jQuery file into the HTML page by mentioning its location, since it is defined as an IIFE, all its methods are readily available for use. Which is why its always written above all the custom / user defined javascript files. 
     (function(){{'{'}}
        alert("Hello World");     
     {{"}"}})()
We dont have to invoke it, it gets automatically executed as soon as it is loaded in the memory.
