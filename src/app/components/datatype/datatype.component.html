Data Types & Variables

The various types of data types present in javascript are :

  • Number
  • String
  • Boolean
  • Undefined
  • Function
  • Object

Javascript is a dynamically typed language, i.e a variable initialized with a number can take a string as a value in the same code block. A javascript variable is declared using the keyword var.

Number The data type number can hold both whole numbers as well as floating/ decimal values. 
      var x = 10; //number

String The data type string can hold pretty much any value. It is defined by putting the string content inside /' or /" quoutes; 
      var str = "Hello World"; //string

Boolean The data type boolean can hold true or false values. 
      var isValid = true; //boolean

Undefined A variable which is declared but not defined will have its value as undefined 
      var x;
      console.log(x); // undefined

Function Functions are blocks of code, which are designed to perform a specific task. The advantage of using functions is, you dont have to write the same code in multiple locations.
